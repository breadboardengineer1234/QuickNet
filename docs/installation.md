# Installation
## Option 1
The easiest way to install QuickNet is to download the latest release here: [Download](https://github.com/breadboardengineer1234/QuickNet/releases)

## Option 2
Copy the [source code](https://github.com/breadboardengineer1234/QuickNet/tree/main/src) into your project. With this method you have to resolve the paths manually.
